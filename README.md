# Timeout Themes

Custom themes for the TimeOut application to make break periods more engaging.

## Installation

To install a theme:

1. Download the theme folder (it should end in `.totheme`).
2. Copy the folder to the `Themes` folder inside your `TimeOut` application directory.
3. Open TimeOut and select the new theme from the application settings.
