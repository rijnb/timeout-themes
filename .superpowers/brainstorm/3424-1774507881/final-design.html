15 Seconds Theme: Final Design

Theme: Retro Digital / Neon | Dissolve: Neon Vaporize

Proposed Architecture

1. Document Structure

  • Themes/15 Seconds.totheme/: Root folder
  • Info.json: Theme metadata
  • Documents/index.html: Main entry point
  • Documents/assets/css/style.css: Layout and Neon styling
  • Documents/assets/js/script.js: Countdown logic and 3D orchestration

2. Visual Components

  • Numbers: Rendered as div elements with SVG or high-impact fonts, styled with multiple text-shadow layers for the "neon glow".
  • 3D Scene: A container with perspective: 1000px. Numbers will use transform: translateZ() to stack in depth.

3. Animation Logic

  • Countdown: Syncs to system clock or a high-frequency interval.
  • The Shift: Every second, the "front" number (e.g., 15) triggers its Vaporize effect (splitting into small DIV "particles" that drift and fade).
  • The Move: Simultaneously, the remaining stack (14, 13, ...) slides forward on the Z-axis using CSS Transitions.
